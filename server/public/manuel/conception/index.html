Structure

Le code source du jeu est organisé avec l'architecture MVC et la programmation objet. Le code est réparti en trois fichiers :

Le fichier controleur : game.py

Ce fichier sert à lancer le jeu et fait le lien avec le modèle et la vue.

Le fichier vue : ihm.py

Ce fichier sert à afficher le jeu (pions, dames, boutons, interface) dans la fenêtre crée précédemment par game.py

Le fichier modèle : data.py

Ce fichier sert à gérer et modifier le représentation interne du jeu, qui est une matrice.
Elle comporte également une classe qui gère l'intelligence artificielle, utilisant l'algorithme minimax.
