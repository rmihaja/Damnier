Conception du jeu

Le jeu à été réalisé avec l'architecture MVC

Le jeu est divisé en trois fichiers

Le premier fichier : Game.py

Ce fichier sert à lancer le jeu et fait le lien avec tkinter

Le deuxième fichier IHM.py:

Ce fichier sert à afficher le jeu(pions, dames, boutons, interface) dans la fenêtre crée précédemment par game.py

Le troisième fichier Data.py:

Ce fichier sert à modifier et gérer les déplacements des pions, dames
