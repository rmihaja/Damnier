Une envie de jouer ?

Essayez notre démo en ligne! (aucune installation requise)

Pour une installation de l'application sur votre ordinateur, suivez ces étapes :

  1. Téléchargez le code source de Damnier
  2. Installez Python 3 ou supérieur (si vous ne l'avez pas encore sur votre machine).
    Télécharger Python 3
  3. La dernière étape consiste à installer socket.io pour la gestion du mode multijoueur.
    Pour cela, ouvrez un terminal et exécutez la commande :

    pip install "python-socketio[client]"

  4. Vous pouvez maintenant lancer le fichier principal avec la commande :

    python chemin-du-dossier-source/app/game.py
