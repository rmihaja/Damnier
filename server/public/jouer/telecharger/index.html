Une envie de jouer ?

Il suffit de vous laissez guider par ces quelques étapes :

1. Tout d'abord notre code et notre jeu est disponible sur le site GitHub

2. Vous pouvez télécharger notre jeu soit sur GitHub ou plus simplement en cliquant juste ici

3. Vous avez besoin de Python 3 pour pouvoir lancer le jeu, si vous ne l'avez pas installer sur votre ordinateur vous pouvez le faire en cliquant ici

4. La dernière étape est d'installer socket.io pour la gestion du mode multijoueur par le biais de cette commande ci dessous

pip install "python-socketio[client]"

5. Vous pouvez maintenant profiter de notre merveilleux jeu en lancer le fichier nommée "Game.py"
